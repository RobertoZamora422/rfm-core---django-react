# Frontend - RFM Core

Aplicación frontend de RFM Core, inicializada con React y Vite como parte de la Fase 1.

## Estado actual

El proyecto se encuentra en inicialización técnica. La aplicación muestra una pantalla institucional mínima y todavía no incluye rutas administrativas, autenticación, servicios HTTP ni pantallas funcionales del sistema.

## Comandos

```powershell
npm install
npm run dev
npm run build
npm run lint
```

## Configuración

El archivo `.env.example` define la URL base prevista para la API:

```text
VITE_API_URL=http://127.0.0.1:8000/api
```

La siguiente etapa corresponde a la construcción del layout administrativo y la integración real con la API cuando las fases backend correspondientes estén disponibles.
